# Plant waterer control

This react app allows you to control a web based plant watering system. The app controls when and how much each plant is watered,
and can present data from soil moisture sensors. This allows remote monitoring, adjustment, and if necessary shut down of the watering system.

### Branches

- `main`: The main branch is the working branch for new features.
- `gh-pages`: The gh-pages branch contains the build files for deployment to GitHub Pages. Actions should automatically deploy new pushed code.
